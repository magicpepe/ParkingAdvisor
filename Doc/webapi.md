	查詢地圖上危險區域及等級

	user's location

	input:
		127.0.0.1/api_1?lat=23.56789&lon=120.213456

        {
          "green": {
            "number": 3,
            "location": [
              "120.644490","24.178780",

            ]
          },
            "yellow": {
            "number": 1,

              "120.645102","24.179225"
